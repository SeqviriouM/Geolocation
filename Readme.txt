В данной программе реализована функция определения маршрута по введенному набору карточек.

В качестве входных данных используются объекты. 

Например, так может выглядеть одна из карточек:
{
  from: Moscow,
  to: Paris,
  transport: airplane,
  number_seat: 15D
}

Для ввода карточки вначале необходимо заполнить поля формы. Обязательными полями являются место отправления и место прибытия. Чтобы добавить к текущей карточки новое поле достаточно нажать кнопку "Добавить поле". В левое поле необходимо ввести название новой информации о маршруте(например "номер рейса", "номер места" и т.д.), а в правое поле его значение.

При нажатии кнопки "Сохранить текущую карточку и ввести еще" происходит сохранение текущей карточки как объекта в массив data и предложение ввода новой карточки.

После окончания ввода всех карточек необходимо нажать кнопку "Расчитать маршрут". После расчета выводится текстовое поле со словесным описание движения.

При нажатии кнопки "Сбросить все" производится перезагрузка страницы.
